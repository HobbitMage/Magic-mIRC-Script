Инструкция по установке Magic Script v.0.7.0

ВНИМАНИЕ!
Скрипт в основном предназначен для новичков, в папке Favorites по умолчанию установлен автоматическим вход на определённые каналы. В большинстве сетей ответы на вопросы об IRC можно получить на канале #help.

1. Установка

Компоненты:
- Файлы скрипта - Содержит необходимые для работы файлы со скриптами
- Настройки скрипта - Текстовые файлы из папки \strings\, которые содержат строковые настройки: причины выхода, каналы и другие.
- Рускоязычная справка - Файл справки от mIRC 6.14, переведенный pirc Team
- Оригинальная справка - файл справки из оригинального дистрибутива

В процессе установки укажите необходимую информацию: ник, почтовый адрес, желаемый сервер, цветовую схему программы.

2. Первый запуск

При запуске программы автоматически открывается окно справки.
*Для изменения цветовой гаммы нажмите Alt+K
*Для изменения информации о себе Alt+O 'Connect'
*Для изменения ведения логов Alt+O 'IRC/Logging' (Chanels - только каналы, Chats - только приваты, Both - все, None - не ведет)

После нажатия кнопки Connect (Левый верхний угол, Молния) вы попадете на сервер по умолчанию, остальные выбираются в меню 'File/Recent servers'.
Для соединения с дополнительным сервером в меню статуса выберите 'Сервер/Имя сервера'

Можете обЧАТься!

3. Применение

Подробнее про применение можно прочитать во встроенной справке.

Использование скрипта не сильно отличается от использования mIRC за небольшим исключением: меню. В появляющихся при правом клике мыши меню есть команды на все случаи чата: 
- Смена сервера и настройки
- Изменение ника и сохранение ника, информация
- Сохранение паролей ника и настроек пользователя, автоматизация идентификации
- Режим away с заготовленными, автоматическими сообщениями, выход, просмотр информации
- Список каналов
- Выход с сервера с сообщениями
- Обращение к пользователям
- Приглашение пользователей на ваши каналы
- Быстрые команды управления каналом
- Отсылание различных личных сообщений пользователям
- Просмотр информации о пользователе
- Контроль над сообщениями, списки игнорирования
- Группы пользователей
- Развитая система действий
- Управление сервисами
- Списки горячих клавиш и символов
- Калькулятор форумных боев
